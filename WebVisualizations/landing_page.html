Summary:

This project was designed to test the effect of latitude on certain weather conditions, specifically temperature, humidity, cloud coverage, and wind speed. We used an OpenWeatherMap API to gather our data, and made the necessary plots using the MatPlotLib library in Python.


In the Visualization pages to the right, key insights are given for each graph concerning how latitude affects each respective weather condition. Source data is provided in the navigation bar, as well as a Comparisons page so that each graph can easily be compared with the others.
